# Oulu Hemodynamics Research Group website

Open `index.html` directly in a browser.

If the browser blocks local files or the page does not update correctly, run a small local server in this folder:

```bash
python -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

The website is static HTML/CSS/JavaScript. No npm install or build step is needed.
